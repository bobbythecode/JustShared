# The Importer
## Features
There abilities as:

* Add partner with asset files.
* View all partners
* Delete partner
* Update partner
* Configurable with config.json file

## Implementation

### Front-end
* Use Flask framework
* Use html templates
* Use the Bootstrap framework

### Back-end
* Use Flask framework
* Implement as RestFUL web service
* SQLite 3
* ORM by Weepee

## Usage
### For users
* View all

```
http://localhost:8000/
```

* Add patner

```
http://localhost:8000/update-partner
```
